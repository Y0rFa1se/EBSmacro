1. properties 폴더의 lectures.ini 파일을 열고 첫줄부터 링크, 아랫줄에 시간 입력 (남은시간 써도됨)

2. 온라인클래스 강의는 66퍼만 들으면 수강완료로 뜨는데 시간 뒤에 ! 붙이면 전체시간 70퍼만 듣고 넘어가짐

+ 남은시간 쓰고 !붙여도됨
-> 강의 전체시간이 3분, 내가 들은시간이 1분이면
남은시간 2분인데 2분의 70퍼(1분 24초)만 들어도 전체 수강시간(2분 24초)은 3분의 70퍼센트(2분 6초)가 넘음
ex) 3:00 -> 2:00!

2. run.exe 실행
3. 마우스 건들지마
4. 창 최대화 하고 듀얼모니터인 경우에는 인터넷 창을 1번 모니터에 두기
5. 실행 후 창 뜨면 로그인 한 후에 커맨드(cmd, 명령 프롬프트)창에 continue입력
6. 안되면 전화해라

※실행되지 않는경우
1. 계정이 다른경우 같은 강의일지라도 링크 주소가 다른 경우가 있음
2. 켜놓고 딴거하면 안됨, 롤하면 한강에 던짐
3. 인터넷이 느려서 명령 입력보다 페이지 로딩이 느리면 중간에 멈출수있음
4. 모니터 해상도가 fhd(1920 x 1080)이 아니면 안될수있음 (아직 안해봄)
5. 맥쓰는사람은 안될걸
6. 제보받음

참고) 인터넷 브라우저는 디폴트로 설정돼있는 브라우저로 나옴
